About This Dashboard

Data Source: Kaggle: Operating Room Utilization Dataset

Methodology:

The original dataset contains 2,170 records. To optimize performance while maintaining statistical validity, a sample size of 327 cases was selected based on a standard sample size calculation (95% Confidence Level, 5% Margin of Error).

Metrics Defined:

  • Start Time: Patient "Wheels In" timestamp.
  • End Time: Patient "Wheels Out" timestamp.
  • Turnover: Calculated gap between cases in the same OR.
